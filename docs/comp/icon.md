# 图标集合

慧泉软件组件库内置图标资源，可通过 CSS 类名直接使用。

## 使用方式

在 HTML 中通过 `<i>` 标签配合对应的类名即可显示图标：

```html
<i class="i_func_30"></i>
<i class="i_arrow_blue"></i>
<i class="i_func_account"></i>
```

## 引入样式

::: tip
项目中已完成引入，如部分页面报错请留意是否未引入样式
:::

在项目中引入图标样式文件：

```html
<link rel="stylesheet" href="/css/icon.css" />
```

或在 JS 中导入：

```js
import '/css/icon.css'
```

<script setup>
import { onMounted } from 'vue'

onMounted(() => {
  document.querySelectorAll('.icon-item').forEach(item => {
    item.addEventListener('click', () => {
      const name = item.querySelector('.icon-name')?.textContent
      if (!name) return
      navigator.clipboard.writeText(name).then(() => {
        const tip = document.createElement('div')
        tip.className = 'copy-tip show'
        tip.textContent = '已复制'
        item.appendChild(tip)
        setTimeout(() => tip.remove(), 1000)
      })
    })
  })
})
</script>
